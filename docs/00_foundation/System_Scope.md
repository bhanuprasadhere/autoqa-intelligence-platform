System Scope Definition
1. In Scope

The system will:

Accept a website URL and credentials

Discover pages and workflows

Identify UI elements such as:

Input fields

Buttons

Dropdowns

Popups

Generate test inputs

Execute tests automatically

Capture screenshots and logs

Generate structured reports

2. Out of Scope (Initial Phase)

The system will NOT:

Bypass CAPTCHA or OTP systems

Hack or exploit unauthorized websites

Replace human decision-making in security approvals

Test native mobile applications

3. Assumptions

The user owns or has permission to test the target website

The website allows automated browser access

Internet connectivity is stable
