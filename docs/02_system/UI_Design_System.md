# UI Design System: "Neon Void"

## 1. Color Palette
* **Background Base:** `#0a0a0f` (Deepest Void Black)
* **Surface:** `#12121a` (Dark Gray-Blue)
* **Primary Accent:** `#00ff9d` (Cyber Green - for Success/Go)
* **Secondary Accent:** `#bc13fe` (Electric Purple - for AI Actions)
* **Error:** `#ff0055` (Neon Red - for Bugs)
* **Text:** `#e0e0e0` (Off-white)

## 2. Typography
* **Headings:** 'Orbitron' or 'Inter' (Bold, uppercase tracking)
* **Body:** 'Inter' or 'Roboto'
* **Code/Logs:** 'Fira Code' or 'JetBrains Mono' (Monospace is crucial for a dev tool)

## 3. Component Styles
* **Cards:** Glassmorphism effect (Translucent background with blur) + 1px Neon Border.
* **Buttons:**
    * *Primary:* Neon Green Glow (`box-shadow: 0 0 10px #00ff9d`).
    * *Secondary:* Hollow outline with Purple border.
* **Animations:**
    * Smooth transitions on hover.
    * "Glitch" effect on critical error headers.
