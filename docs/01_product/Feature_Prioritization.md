# Feature Prioritization Matrix (MoSCoW)

## Must Have (MVP - The "Walking Skeleton")
* User Login/Sign-up (Supabase Auth).
* Input URL & Start Scan.
* Basic Crawler (Finds links and forms).
* Basic Negative Testing (Empty fields, bad data).
* JSON Report Output.
* Screenshot on Error.

## Should Have (V1.0 - The "Market Ready" Product)
* Neon UI Dashboard with Charts.
* "Abuse Testing" (Rapid clicks).
* History/Comparison of past runs.
* Email alerts on critical bugs.
* Handling of simple Login flows using provided credentials.

## Could Have (V2.0 - The "Enterprise" Version)
* AI-powered "Summary of Bugs" (Natural Language).
* Integration with Jira/GitHub Issues.
* Visual Regression Testing (Compare pixels).
* Custom headers/cookies configuration.

## Won't Have (Right Now)
* Mobile App Native Testing (.apk/.ipa).
* CAPTCHA Solving.
* DDoS simulation (Legally dangerous).
