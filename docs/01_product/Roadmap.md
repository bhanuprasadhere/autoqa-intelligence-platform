# Product Roadmap

## Phase 1: Foundation (Weeks 1-2)
* **Goal:** Setup Tech Stack & "Hello World" Crawler.
* **Tasks:**
    * Initialize Repo & Docs.
    * Setup Next.js + Neon UI Framework.
    * Setup Supabase Project.
    * Build "Crawler Agent" prototype that just lists all URLs on a site.

## Phase 2: The Tester Brain (Weeks 3-5)
* **Goal:** Make the crawler intelligent.
* **Tasks:**
    * Implement Form Detection logic.
    * Connect LLM (Gemini/Claude) to generate test data.
    * Build the "Abuse" logic (button mashing).
    * Implement Screenshot capture.

## Phase 3: Reporting & Polish (Weeks 6-8)
* **Goal:** Make it usable for humans.
* **Tasks:**
    * Build the Dashboard (Charts, Logs).
    * Implement JSON Export.
    * Refine the Neon UI design.
    * End-to-End testing of the tool itself.
